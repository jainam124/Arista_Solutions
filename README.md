## Install packages and dependencies

``` npm init -y ```

``` npm install express ```

``` npm install -D tailwindcss postcss autoprefixer ```

``` npx tailwindcss init ```

``` npm i --save-dev nodemon ```

``` npm install mysql ```

``` npm install multer path fs ejs ```

``` npm install stripe```
